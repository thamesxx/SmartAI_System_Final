## 🎯 Common Tasks

### Task: Get Started Quickly
1. Read [GETTING_STARTED.md](GETTING_STARTED.md)
2. Run `./start-dev.sh` or `start-dev.bat`
3. Open http://localhost:3000

### Task: Understand the API
1. Read [API_QUICK_REFERENCE.md](API_QUICK_REFERENCE.md)
2. Review [fastapi_backend_example.py](fastapi_backend_example.py)
3. Visit http://localhost:8000/docs

### Task: Connect Real Data
1. Read [BACKEND_API_DOCUMENTATION.md](BACKEND_API_DOCUMENTATION.md)
2. Edit [fastapi_backend_example.py](fastapi_backend_example.py)
3. Connect your database/data source

### Task: Deploy to Production
1. Read [SETUP_GUIDE.md](SETUP_GUIDE.md) → Production section
2. Configure [.env.example](.env.example) → `.env`
3. Build frontend: `npm run build`
4. Run backend with gunicorn

### Task: Customize Frontend
1. Review [services/api.ts](services/api.ts) for API calls
2. Edit components in `/components` directory
3. Modify refresh rates in components

### Task: Add New Endpoint
1. Add endpoint to [fastapi_backend_example.py](fastapi_backend_example.py)
2. Add function to [services/api.ts](services/api.ts)
3. Use in component

### Task: Troubleshoot Issues
1. Check [GETTING_STARTED.md](GETTING_STARTED.md) → Troubleshooting
2. Check [SETUP_GUIDE.md](SETUP_GUIDE.md) → Troubleshooting
3. Review browser console and backend logs
